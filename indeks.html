Penjelasan Kode Kivy

1. Import Library

Mengimpor App, Button, GridLayout dari Kivy.

2. Kelas MyApp

MyApp adalah kelas utama yang mewarisi App.

3. Fungsi build()

build() membuat layout dan tombol-tombol.

4. Membuat GridLayout

Membuat tata letak grid dengan 3 kolom.

5. Membuat Tombol

Button dengan teks dan ukuran tertentu.

6. Menambahkan ke Layout

add_widget() menambahkan tombol ke GridLayout.

7. Menjalankan App

MyApp().run() menjalankan aplikasi.
