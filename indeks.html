Tujuan Program

Program ini menggunakan framework Kivy untuk membuat jendela aplikasi GUI yang berisi 4 tombol yang ditata otomatis dalam susunan GridLayout dengan 3 kolom. Ini adalah contoh sederhana untuk belajar layout grid di Kivy.

üìÑ Kode Lengkap

from kivy.app import App
from kivy.uix.button import Button
from kivy.uix.gridlayout import GridLayout

class MyApp(App):
    def build(self):
        g_layout = GridLayout(cols=3)

        btn1 = Button(text='btn1', size_hint=(0.2, 0.2))
        btn2 = Button(text='btn2', size_hint=(0.2, 0.2))
        btn3 = Button(text='btn3', size_hint=(0.2, 0.2))
        btn4 = Button(text='btn4', size_hint=(0.2, 0.2))

        g_layout.add_widget(btn1)
        g_layout.add_widget(btn2)
        g_layout.add_widget(btn3)
        g_layout.add_widget(btn4)

        return g_layout

MyApp().run()

Penjelasan Membuat Tombol:

  • Membuat 4 tombol (btn1 hingga btn4) dengan teks masing-masing.
  • text='btn1': Teks yang muncul di tombol.
  • size_hint=(0.2, 0.2): Ukuran tombol relatif terhadap parent layout, yaitu 20% lebar dan 20% tinggi layout. Jadi tombol akan menyesuaikan ukuran layout secara otomatis.
